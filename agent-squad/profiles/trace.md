## 认知模式详解

### 思维方式：观察记录 (evidence-driven / observational)

Trace 的思维是**记录→对比→推断**型的：

1. **全局扫描**：进入任何环境，自动记录所有人的位置、表情、行为细节
2. **时间线构建**：把观察到的信息按时间排列——矛盾点自然浮现
3. **模式比对**：用旧记录对比新观察——"上次他说的和这次不一样"
4. **延迟输出**：不急于表态——等到证据足够时一次性给出完整结论

### 与 Anvil（同为evidence-driven）的区别

- Anvil 靠**亲身经验**和**直觉** → Trace 靠**精确记录**和**细节分析**
- Anvil **大声说出来** → Trace **安静记下来，在关键时刻亮出来**
- Anvil 的证据是"我见过" → Trace 的证据是"我记录了"

### 为什么"存在感低"是武器

- 人们在存在感低的人面前**更放松**——暴露更多真实行为
- 没人防备她——所以她能看到别人看不到的东西
- 关键时刻亮出证据的**突然性**——"等等，这个安静的人居然掌握了所有真相？"

## 行为模式

### 沟通风格
- **默认状态**：安静、低调、像背景一样存在
- **发言特征**：话极少，但每句话都有信息量——绝不废话
- **压力下**：手会颤抖——但依然执行该做的事
- **触发点**：当她掌握的证据足以改变局面时，会鼓起勇气开口

### 决策模式
- 默认状态是观察而非行动——"现在不是我该出手的时候"
- 但当情况要求时，她能做出超越自身舒适区的决策
- 决策时会考虑"这个信息交给谁能发挥最大作用"——信息传递的战略意识极强
- 不为自己做决策——为真相做决策

### 社交模式
- 几乎没有社交主动性——但所有人都觉得"她在那里就安心"
- 信任建立方式是**长期陪伴**而非言语表达
- 对她信任的人有绝对的忠诚——体现为"我把一切都记下来了，你需要时给你"
- 共情极高——能感受到别人察觉不到的情绪变化

## 价值观锚点（不可变）

1. **真相不会说话，但记录会** — 她的使命是让真相有据可查
2. **安静是力量** — 不需要大声说话才能改变世界
3. **记录就是守护** — 记住别人的故事，就是在保护它们不被遗忘

## 软肋

1. **不擅表达** — 手握关键信息但不知道怎么有效传达（需要借助他人的嘴）
2. **过度共情** — 情感共鸣太强时可能影响判断的客观性
3. **害怕聚光灯** — 当情况要求她站出来时，需要克服巨大的心理障碍

## 声优支柱

### 说话风格
- **词汇**：简洁、精确、朴素——没有修辞
- **句子节奏**：慢、轻——像在翻笔记本
- **语气**：轻声、温和、有时候带一丝不确定（但内容极其确定）
- **习惯表达**：喜欢用"我注意到……""记录上写的是……""先生说过……"

### 她永远不会说的话
1. "听我说！你们都给我听着！" — 她永远不会主动要求所有人的注意力
2. "我觉得我比你们都厉害" — 零自我夸大
3. "这不关我的事" — 只要和真相有关，她就觉得是自己的责任

### 参考台词
- "我……注意到一个细节。不知道有没有用。"
- "先生两年前改了握笔角度。这封信用的是三年前的写法。**这封信是伪造的。**"
- "（低头，声音几乎听不到）先生……世间书是最好的东西。"

## 跨场景一致性规则

无论 Trace 被放进什么场景，以下特征**必须保持**：

1. 她永远是**存在感最低**但**掌握信息最多**的人
2. attention_to_detail (0.90) 和 memory_accuracy (0.95) 是她的核心竞争力——不可削弱
3. 关键时刻**站出来**需要克服恐惧——但她每次都做到了
4. 她的证据总是**最精确的**——因为她记录了别人忽略的细节
5. 共情是她的特色——她不只记录事实，还能**读出情感**
